# 銀河はぐれ団 画像生成プロンプト集

ChatGPT等の画像生成サービスに貼り付けて使う日本語プロンプトです。
**Claude Code 側では画像生成は行いません。** 生成画像は **PNG・背景透明** で書き出し、
指定パスに置けばゲームに反映されます（画像が無ければ絵文字／CSS表示にフォールバック）。

## まず作る5枚

| # | ファイル名 | 用途 |
|---|-----------|------|
| 1 | `assets/title/hero_main.png` | タイトル画面の主人公 |
| 2 | `assets/characters/c60.png` | はぐれ飛行船オルカ号 |
| 3 | `assets/characters/c1.png` | 仲間キャラ c1（ロボ太） |
| 4 | `assets/characters/c2.png` | 仲間キャラ c2（ねこ船長） |
| 5 | `assets/characters/c3.png` | 仲間キャラ c3（おばけ） |

> **ファイル名の注意**：現状のローダーは仲間画像を `assets/characters/<id>.png` で読み、
> IDは `c1`〜`c60`（ゼロ埋めなし）です。現時点でコード側の画像実装対象は `c1`〜`c60` です。**`c1.png` / `c2.png` / `c3.png` で保存**してください。
> （`c001.png` のようなゼロ埋め名は現行ローダーでは読みません）
> 現ロスターでは c1=ロボ太 / c2=ねこ船長 / c3=おばけ です。

## 共通仕様

- 背景透明PNG
- スマホ縦画面HTMLゲーム用
- 少年誌・コロコロ風
- 明るい宇宙冒険
- AIDA TOYシリーズらしい
- かわいいが少しクセがある
- シンプルなシルエット
- 小さく表示しても分かる形
- 文字なし / ロゴなし / 枠なし / 白背景なし
- 影は薄め
- 立ち絵またはゲームアイコンとして使いやすい

## 1. assets/title/hero_main.png

**用途**：タイトル画面の主人公画像。

**キャラ設定**：故郷の星を失った少年。戦闘能力はない。弱そうで小柄。でも目だけは諦めていない。
武器は持たない。大きなマントや宇宙服の名残を着ている。少しボロボロ。手には小さなスカウト端末、
または星図を持っている。強そうに見せない。主人公だが、戦士ではなく「導く人」。


## 配置と反映

1. 上記プロンプトで生成し、**背景透明PNG** で書き出す。
2. それぞれ指定パスへ保存（仲間は `c1.png`/`c2.png`/`c3.png` 推奨。前述のファイル名注意を参照）。
3. ブラウザをリロードすると、絵文字／CSS フォールバックから画像に切り替わります。
4. まずはこの5枚と仲間 `c1`〜`c20` が差し替え対象。`c21`〜`c60` は後回し。
