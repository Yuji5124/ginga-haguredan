# 銀河はぐれ団 画像生成プロンプト集

ChatGPT等の画像生成に貼り付けて使う日本語プロンプトです。
生成画像は **PNG・背景透明** で書き出し、指定パスに置けばゲームに反映されます。

## 共通スタイル指定（各プロンプトの先頭に付ける）

```
スマホゲーム用のキャラクター画像。
画風：少年誌・コロコロコミック風、明るい宇宙冒険もの、AIDA TOYシリーズらしいかわいさ。
かわいいが少しクセがある。太めの主線、はっきりした配色、シンプルで分かりやすいシルエット。
背景は完全に透明（PNG）。影や複雑な背景は描かない。被写体を中央に、全身が収まるように。
```

## 配置と反映

1. 上記プロンプトで生成し、**背景透明PNG**で書き出す。
2. それぞれ指定パスへ保存：
   - `assets/title/hero_main.png`
   - `assets/ships/ship_default.png`
   - `assets/characters/c1.png` / `c2.png` / `c3.png`
3. ブラウザをリロードすると、絵文字フォールバックから画像に切り替わります。

> 仲間キャラのファイル名は `app.js` の ID（`c1`〜`c100`）に合わせて `c<番号>.png`。
> 敵は `assets/enemies/e<番号>.png`（`e1`〜`e20`）。
